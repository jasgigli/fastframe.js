# Changelog

All notable changes to FastFrame.js will be documented in this file.

## [Unreleased]

### Added
- Improved package configuration for better NPM publishing
- Added Node.js engine requirements
- Enhanced release workflow with automatic changelog generation

## [0.1.0] - 2023-04-15

### Added
- Initial release of FastFrame.js
- Core reactivity system with signals and effects
- Component-based architecture with .ff.js files
- Compiler with Vite plugin for transforming components
- CLI for creating and building projects
- Webpack loader for webpack integration
- Rollup plugin for rollup integration
- Documentation site with VitePress
- Example application showcasing framework features
